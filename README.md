# HospitalManagementSystem


<h2>Migration</h2>
[Package Manager console]
<br />
add-migration &ltname of migration&gt
<br />
<br />
[Command Line] <br />
change directory to .proj file
<br />
run script below -->
<br />
dotnet ef migrations add &ltname of migration&gt
<br />


<h2>Run</h2>
change directory to .proj file
<br />
dotnet watch run
<br />
